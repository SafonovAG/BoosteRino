Используйте этот метод для получения информации о заказе

Пример запроса

https://twiboost.com/api/v2?action=status&order=1&key=yourKey

Пример ответа

{
  "charge": "0.27819",
  "start_count": "3572",
  "status": "Partial",
  "remains": "157",
  "currency": "USD"
}

Метод возвращает объект, содержащий следующие поля:

charge

Double

Потраченные на заказ деньги

start_count

Integer

Количество на момент активации заказа

status

String

Статус задания. Одно из

In progress

Completed

Awaiting

Canceled

Fail

Partial

remains

Integer

Оставшееся количество

currency

String

Валюта заказа
