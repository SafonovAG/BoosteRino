Используйте этот метод для получения баланса аккаунта

Пример запроса

https://twiboost.com/api/v2?action=balance&key=yourKey

Пример ответа

{
  "balance": "99.80",
  "currency": "USD"
}

Метод возвращает объект, содержащий следующие поля:

balance

String

Текущий баланс

currency

String

Валюта баланса
