# BoosteRino

Веб-приложение для SMM-продвижения созданное полностью на Cursor

## Стек

- PHP 8.3 + Apache
- MySQL 8
- Vanilla JavaScript

## Установка

1. `composer install`
2. Скопировать `config/database.example.php` → `config/database.php`
3. Импорт `database/schema.sql`
4. `php bin/create_superadmin.php`
5. DocumentRoot Apache → `public/`
